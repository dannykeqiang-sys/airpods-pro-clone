## 发现的差异汇总

### 需修复的差异（中等严重度）

1. **Hero 购买区域布局**：原站价格+购买按钮在右下角独立浮窗（marquee-detail），复刻站在文字下方。需要调整 HeroSection 的购买区域布局。

2. **Hero 产品图位置**：原站产品图在文字右侧独立展示（与文字区域分离），复刻站产品图作为视频背景覆盖全区域。需要调整 Hero 的图文布局关系。

### 可接受的差异（低严重度）

3. **导航栏下拉菜单**：原站桌面端 hover 展开完整下拉菜单，复刻站仅实现了移动端菜单。桌面端下拉菜单是次要功能。

4. **本地导航栏**：原站有本地导航栏（AirPods Pro 子页面导航），复刻站未实现。这是次要功能。

5. **脚注系统**：原站有完整的脚注弹窗系统，复刻站仅有脚注链接。这是次要功能。

6. **页脚**：原站有完整的 Apple 页脚（多列目录+法律条款），复刻站仅有简化版页脚。

## 修复执行

### 修复 #1: Hero 购买区域布局 ✅ 已修复
- 将价格+购买按钮从文字下方移到右下角独立浮窗
- 使用 `position: absolute; bottom: 80px; right: calc((100% - 980px) / 2 + 22px)` 定位
- 与原站 marquee-detail 布局一致

### 修复 #2: Hero 产品图位置 ✅ 无需修复
- 原站实际上也是视频覆盖全区域，产品图是视频内容的一部分
- 视频背景是正确的实现方式，与原站行为一致
