# Contrast Section 规格文档

## 概述
"看看哪款 AirPods 适合你"对比区域，展示 AirPods Pro 3 vs AirPods 4 (ANC)。

## 布局结构
- Headline: H2 "看看哪款 AirPods 适合你"
- CTA: "探索全部 AirPods 机型"
- 2 个 product-tile 对比卡片
- 每个卡片: 产品图 + 名称 + 定位描述 + 价格 + 功能列表 + 购买按钮
- 底部: "比较各款 AirPods 机型" CTA

## 内容数据
### AirPods Pro 3
- 定位: "入耳式主动降噪，入主新声代；具备心率传感功能..."
- 价格: RMB 1899
- 功能: 主动降噪4倍⁹、自适应音频¹¹、空间音频²、心率传感³、降低高音量、8小时⁵、语音突显¹⁰

### AirPods 4 (ANC)
- 定位: "音质、舒适性和噪声控制，再进阶"
- 价格: RMB 1399
- 功能: 主动降噪¹¹、自适应音频¹¹、空间音频²、无心率传感、无降低高音量、5小时¹⁹、语音突显¹⁰
