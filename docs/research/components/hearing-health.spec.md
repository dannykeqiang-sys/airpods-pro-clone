# Hearing Health Section 规格文档

## 概述
"为听力护航"区域，展示降低高音量、对话增强和背景音功能。

## 布局结构
- Eyebrow: H2 "为听力护航" (typography-eyebrow-elevated)
- Headline: H3 "强噪音，尽可能减轻。" (typography-headline-standalone)
- Hero: 生活方式图 + 增强图 (hearing_health_lifestyle → hearing_health_lifestyle_enhanced)
- 功能图: hearing_health_endframe (490x490)

## 内容数据
- "降低高音量功能，会尽可能为你减弱强噪音。对话增强功能，让你专注聆听面前说话者的声音。背景音功能，可播放海浪或下雨等舒缓的声音..."

## 样式
- 背景: #f5f5f7 (background-alt)
- 图片居中展示
- 文字在图片下方

## 交互行为
- 滚动触发图片过渡动画 (lifestyle → enhanced)
- 功能图标渐显
