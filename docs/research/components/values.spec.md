# Values Section 规格文档

## 概述
"我们的价值观，为我们导航"区域，展示环境责任、隐私和辅助功能。

## 布局结构
- Headline: H2 "我们的价值观，为我们导航。"
- 3 个 icon-card (图标 + headline + body + CTA链接)

## 内容数据
1. 环境责任 - "我们的计划，和我们的产品一样突破创新" + "我们致力于到 2030 年实现整体碳足迹净零排放" → "进一步了解环境责任"
2. 隐私安全 - "力保个人信息安全，这很 Apple" + "隐私是一项基本人权..." → "进一步了解隐私安全"
3. 辅助功能 - "种种创新，为辅助使用而设计" + "我们的产品和服务为每一个人而设计..." → "进一步了解辅助功能"

## 样式
- 背景: #f5f5f7 (background-alt)
- 3 个 icon-card 横向排列
- 每个卡片: 图标 + headline + body + 链接

## 交互行为
- 滚动触发 staggered-start 动画
- CTA 链接带箭头图标
